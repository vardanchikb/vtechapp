Shared Folder Permission Viewer v1.0.0

Initial public Windows release.

- Recursive NTFS permission scanning for shared folders and nested directory trees.
- Permission explorer for explicit and inherited access control entries.
- Security analysis views for inheritance overrides, deny rules, privilege review, unauthorized entries, orphaned SIDs, and user search.
- Local scan session database for reopening audit results.
