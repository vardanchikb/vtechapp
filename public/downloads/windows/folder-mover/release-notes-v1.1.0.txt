Folder Mover 1.1.0

Folder Mover 1.1.0 adds a new NTFS permission copy mode named Preserve Inheritance.
This mode is built for server, domain, and file-share migrations where the destination
must keep the same permission inheritance structure as the source tree.

What's new

- Added Preserve Inheritance permission mode.
- Preserves source ACL inheritance structure across all folder levels, not only the root.
- Copies explicit permissions at each file and folder level, then lets child items inherit
  from their matching copied destination parent.
- Seeds the copied root folder with the source root's effective permissions so the copied
  tree has the correct inheritance starting point.
- Works for both Copy and Move operations.
- Clarifies that Merge with Inherited remains an additive mode: it keeps destination
  inheritance enabled, but source effective permissions are added explicitly.
